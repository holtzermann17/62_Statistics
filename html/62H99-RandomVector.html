random vector


A random vector is a finite-dimensional formal vector of random variables. The random vector can be written either as a column or row of random variables, depending on its context and use. So if X1,X2,…,Xn are random variables, then

𝐗=(X1X2⋮Xn)=(X1,X2,…,Xn)T

is a random (column) vector. Similarly, one defines a random matrix to be a formal matrix whose entries are all random variables. The size of a random vector and the size of a random matrix are assumed to be finite fixed constants.

The distribution of a random vector 𝐗=(X1,X2,…,Xn) is defined to be the joint distribution of its coordinates X1,…,Xn:

F𝐗⁢(𝐱):=FX1,…,Xn⁢(x1,…,xn).

Similarly, the distribution of a random matrix is the joint distribution of its matrix components.

Let 𝐗=(X1,X2,…,Xn) be a random vector. If E⁡[Xi] exists (<∞) for each i, then the expectation of X, called the mean vector and denoted by 𝐄⁢[𝐗], is defined to be:

𝐄⁢[𝐗]:=(E⁡[X1],E⁡[X2],…,E⁡[Xn]).

Clearly 𝐄⁢[𝐗]T=𝐄⁢[𝐗T]. The expectation of a random matrix is similarly defined. Note that the definitions of expectations can also be defined via measure theory. Then, using Fubini’s Theorem, one can show that the two sets of definitions coincide.

Again, let 𝐗=(X1,X2,…,Xn)T be a random vector. If 𝝁=𝐄⁢[𝐗] is defined and E⁡[Xi⁢Xj] are defined for all 1≤i,j≤n, then the variance of X, denoted by 𝐕𝐚𝐫⁢[𝐗], is defined to be:

𝐕𝐚𝐫⁢[𝐗]:=𝐄⁢[(𝐗-𝝁)⁢(𝐗-𝝁)T].

It is not hard to see that 𝐕𝐚𝐫⁢[𝐗] is an n×n symmetric matrix and it is equal to the covariance matrix of the Xi’s.

:

  1. 1.

    If X is an n-dimensional random vector with A a m×n constant matrix and 𝜶 an m-dimensional constant vector, then

    𝐄⁢[𝐀𝐗+𝜶]=𝐀𝐄⁢[𝐗]+𝜶.
  2. 2.

    Same set up as above. Then

    𝐕𝐚𝐫⁢[𝐀𝐗+𝜶]=𝐀𝐕𝐚𝐫⁢[𝐗]⁢𝐀T.

    If the Xi’s are iid (independent identically distributed), with variance 𝝈2, then

    𝐕𝐚𝐫⁢[𝐀𝐗+𝜶]=𝝈2⁢𝐀𝐀T.
  3. 3.

    Let 𝐗 be an n-dimensional random vector with 𝝁=𝐄⁢[𝐗], 𝚺=𝐕𝐚𝐫⁢[𝐗]. 𝐀 is an n×n constant matrix. Then

    𝐄⁢[𝐗T⁢𝐀𝐗]=tr⁡(𝐀⁢𝚺)+𝝁T⁢𝐀⁢𝝁.
