hypergeometric random variable


X is a hypergeometric random variable with parameters M,K,n if

fX⁢(x)=(Kx)⁢(M-Kn-x)(Mn), x={0,1,…,n}

Parameters:

  • ⋆

    M∈{1,2,…}

  • ⋆

    K∈{0,1,…,M}

  • ⋆

    n∈{1,2,…,M}

Syntax:

X∼H⁢y⁢p⁢e⁢r⁢g⁢e⁢o⁢(M,K,n)

Notes:

  1. 1.

    X represents the number of “special” items (from the K special items) present on a sample of n from a population with M items.

  2. 2.

    The expected value of X is noted as E⁢[X]=n⁢KM

  3. 3.

    The variance of X is noted as V⁢a⁢r⁢[X]=n⁢KM⁢M-KM⁢M-nM-1

Approximation techniques:

If (K2)≪n,M-K+1-n then X can be approximated as a binomial random variable with parameters n=K and p=M-K+1-nM-K+1. This approximation simplifies the distribution by looking at a system with replacement for large values of M and K.
