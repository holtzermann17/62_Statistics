Fisher information matrix


Given a statistical model {f𝐗(𝒙∣𝜽)} of a random vector X, the Fisher information matrix, I, is the variance of the score function U. So,

I=Var⁡[U].

If there is only one parameter involved, then I is simply called the Fisher information or information of f𝐗(𝒙∣θ).

Remarks

  • •

    If f𝐗(𝒙∣𝜽) belongs to the exponential family, I=E⁡[UT⁢U]. Furthermore, with some regularity conditions imposed, we have

    I=-E⁡[∂⁡U∂⁡𝜽].
  • •

    As an example, the normal distribution, N⁢(μ,σ2), belongs to the exponential family and its log-likelihood function ℓ(𝜽∣x) is

    -12⁢ln⁡(2⁢π⁢σ2)-(x-μ)22⁢σ2,

    where 𝜽=(μ,σ2). Then the score function U⁢(𝜽) is given by

    (∂⁡ℓ∂⁡μ,∂⁡ℓ∂⁡σ2)=(x-μσ2,(x-μ)22⁢σ4-12⁢σ2).

    Taking the derivative with respect to 𝜽, we have

    ∂⁡U∂⁡𝜽=(∂⁡U1∂⁡μ∂⁡U2∂⁡μ∂⁡U1∂⁡σ2∂⁡U2∂⁡σ2)=(-1σ2-x-μσ4-x-μσ412⁢σ4-(x-μ)2σ6).

    Therefore, the Fisher information matrix I is

    -E⁡[∂⁡U∂⁡𝜽]=12⁢σ4⁢(2⁢σ200-1).
  • •

    Now, in linear regression model with constant variance σ2, it can be shown that the Fisher information matrix I is

    1σ2⁢𝐗T⁢𝐗,

    where X is the design matrix of the regression model.

  • •

    In general, the Fisher information meansures how much “information” is known about a parameter θ. If T is an unbiased estimator of θ, it can be shown that

    Var⁡[T⁢(X)]≥1I⁢(θ)

    This is known as the Cramer-Rao inequality, and the number 1/I⁢(θ) is known as the Cramer-Rao lower bound. The smaller the variance of the estimate of θ, the more information we have on θ. If there is more than one parameter, the above can be generalized by saying that

    Var⁡[T⁢(X)]-I⁢(𝜽)-1

    is positive semidefinite, where I is the Fisher information matrix.
