Linear Plexing


We have fixed α as the maximum acceptable probability of commiting a type I error. But associated with each normalized test of significance is γ, the probability of commiting a type I error.

Notice that as we make γ smaller, the smaller the critical region becomes and thus the larger the value of the test statistic must be for the null hypothesis to be rejected.

Find p, defined as the smallest γ for which we can still reject H0.

Reject H0 if α<p.
