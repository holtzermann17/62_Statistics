negative hypergeometric random variable


X is a negative hypergeometric random variable with parameters W,B,b if

fX⁢(x)=(x+b-1x)⁢(W+B-b-xW-x)(W+BW), x={0,1,…,W}

Parameters:

  • ⋆

    W∈{1,2,…}

  • ⋆

    B∈{1,2,…}

  • ⋆

    b∈{1,2,…,B}

Syntax:

X∼N⁢e⁢g⁢H⁢y⁢p⁢e⁢r⁢g⁢e⁢o⁢(W,B,b)

Notes:

  1. 1.

    X represents the number of “special” items (from the W special items) present before the bth object from a population with B items.

  2. 2.

    The expected value of X is noted as E⁢[X]=W⁢bB+1

  3. 3.

    The variance of X is noted as V⁢a⁢r⁢[X]=W⁢b⁢(B-b+1)⁢(W+B+1)(B+2)⁢(B+1)2

Approximation techniques:

If (x2)≪W and (b2)≪B then X can be approximated as a negative binomial random variable with parameters r=b and p=WW+B. This approximation simplifies the distribution by looking at a system with replacement for large values of W and B.
