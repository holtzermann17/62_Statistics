score function


Given a statistical model {f𝐗(𝒙∣𝜽):𝜽∈Θ} with log-likelihood function ℓ(𝜽∣𝒙), the score function U is defined to be the gradient of ℓ:

U⁢(𝜽)=∇⁡ℓ=∂⁡ℓ∂⁡𝜽.

Since the score function U is also a function of the random vector 𝒙, U is itself a random vector. By setting U to 0, we have a system of k equation(s), otherwise known as the likelihood equation(s):

U⁢(𝜽)=(∂⁡ℓ∂⁡θ1,…,∂⁡ℓ∂⁡θk)=(0,…,0).

If 𝜽=θ is one-dimensional, then the score function is simply referred to as the score of θ.

The maximum likelihood estimate (MLE) 𝜽^ of the parameter vector 𝜽 can usually be found by finding the solutions of the likelihood equations. The likelihood equations may also be formed by setting the gradient of the plain likelihood function to zero. The use of the log function often facilitates the algebra as many distributions are exponential in nature. For some distributions it may also be necessary to test that the solution to the likelihood equations is really a minimum as opposed to a point of inflection.

Example. n independent observations are made from a random variable X with a Poisson distribution with parameter λ. The observed values are x1,…,xn. The log-likelihood of the joint pdf is

ℓ(λ∣𝒙)=∑i=1n-λ+xiln(λ)-ln(xi!)

and so the score function is

U⁢(λ)=d⁢ℓd⁢λ=∑i=1n(-1+xiλ)=-n+n⁢x¯λ,

where n⁢x¯=∑xi. To find the MLE of λ, we set U=0 and solve for λ. So the MLE λ^ of λ=x¯.
