lognormal random variable


X is a lognormal random variable with parameters μ∈ℝ and σ2>0 if its probability density function is given for x>0 by

fX⁢(x)=12⁢π⁢σ2⁢e-(ln⁡x-μ)22⁢σ2x.

To denote this, one usually writes X∼L⁢o⁢g⁢N⁢(μ,σ2).

For a lognormal random variable X:

  1. 1.

    X is a random variable such that ln⁡(X) is a normal random variable with mean μ and variance σ2.

  2. 2.

    E⁢[X]=eμ+σ2/2

  3. 3.

    V⁢a⁢r⁢[X]=e2⁢μ+σ2⁢(eσ2-1)

  4. 4.

    MX⁢(t) is not a useful quantity.
