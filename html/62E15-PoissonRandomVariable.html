Poisson random variable


The Poisson discrete probability function with parameter λ>0 is given by

fX⁢(x)=e-λ⁢λxx!,x∈ℕ.

A random variable X with such a density has expectation, variance, moment generating function and characteristic function given by E⁢[X]=λ, V⁢a⁢r⁢[X]=λ, MX⁢(t)=eλ⁢(et-1), and ϕX⁢(t)=eλ⁢(ei⁢t-1), respectively.
