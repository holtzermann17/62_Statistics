geometric random variable


A geometric random variable with parameter p∈(0,1] is one whose density distribution function is given by

fX⁢(x)=p⁢(1-p)x,x=0,1,2,…

This is denoted by X∼G⁢e⁢o⁢(p).

Notes:

  1. 1.

    A standard application of geometric random variables is where X represents the number of failed Bernoulli trials before the first success.

  2. 2.

    The expected value of a geometric random variable is given by E⁢[X]=1-pp, and the variance by V⁢a⁢r⁢[X]=1-pp2

  3. 3.

    The moment generating function of a geometric random variable is given by MX⁢(t)=p1-(1-p)⁢et.
